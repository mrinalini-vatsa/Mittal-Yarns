# Mittal Yarns Management System

A full-stack yarn industry management system built using React, Node.js, Prisma, and PostgreSQL.

## Features
- Worker management (Supervisor / Operator / Spinner)
- Role-based access control (Admin vs Supervisor)
- Attendance tracking with export to Excel
- Inventory management with controlled access
- Order management (Admin only)
- Inquiry system with email integration

## Tech Stack
- Frontend: React + Vite + Tailwind
- Backend: Node.js + Express
- Database: Prisma ORM
- Auth: JWT-based authentication
